## 1. 产品概述
多人在线虚拟时尚设计工作室与全球时装秀联赛系统 — 玩家创建并运营个人时尚工作室，招募设计师/模特/裁缝团队，设计制作服装，参与每日时装秀联赛，通过交易和升级打造全球顶级时尚品牌。
- 目标用户：热爱时尚、设计、模拟经营类游戏的在线玩家群体
- 核心价值：将时尚设计与竞技联赛融合，提供沉浸式创作+竞争+社交体验

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 工作室创始人 | 游戏注册 | 创建工作室、设定风格、全权管理 |
| 工作室成员 | 被邀请加入 | 设计/制作/走秀等分工操作 |
| 访客 | 无需注册 | 浏览排行榜、观看时装秀 |

### 2.2 功能模块
1. **总控台页面**：工作室概览、今日待办、实时动态、快捷入口
2. **工作室管理页面**：创建工作室、设定风格（古风/赛博/自然）、布局展示、员工管理
3. **设计工坊页面**：选面料/颜色/配饰、时尚指数计算、稀有词缀触发、特效展示
4. **制作车间页面**：材料消耗、成功率计算、品质判定、失败返部分材料
5. **时装秀页面**：每日联赛、实时观众情绪、模特状态、后台事件、技能使用
6. **交易中心页面**：定价出售、系统建议区间、成交公告、潮流风向事件
7. **升级扩建页面**：设施升级、权限审批、首席/副首席/财务主管三级体系
8. **运营报告页面**：热度热力图、成长曲线、收入趋势、雷达图PDF导出
9. **排行榜页面**：全服排名、他人工作室查看、作品库浏览

## 3. 核心流程

### 3.1 设计制作流程
玩家进入设计工坊 → 选择面料/颜色/配饰 → 系统实时计算时尚指数和稀有词缀触发概率 → 确认设计稿 → 进入制作车间 → 系统根据裁缝技能和设备等级计算成功率 → 制作判定 → 成功：品质评定，进入库存；失败：返还部分材料

### 3.2 时装秀联赛流程
每日定时开赛 → 系统按实力自动匹配对手和评审团 → 走秀开始 → 实时更新观众情绪/模特状态 → 突发事件触发 → 玩家手动使用技能应对 → 走秀结束 → 按时尚指数+投票+评委分综合排名 → 发放积分和稀有面料图纸

### 3.3 交易流程
卖家选择成衣/设计图 → 设定价格（系统建议区间基于近7天同类成交均价）→ 上架 → 买家浏览购买 → 成交全服公告 → 可能触发"潮流风向"事件

## 4. 用户界面设计

### 4.1 设计风格
- **主题风格**：奢华时尚杂志风 + 赛博朋克数据可视化，融合高级时装编辑部的精致感和游戏化交互
- **主色调**：深邃黑(#0A0A0F)为底，金色(#D4AF37)为主强调色，品红(#FF2D78)为次强调色
- **辅助色**：冰蓝(#00D4FF)用于数据/科技感元素，柔紫(#B388FF)用于稀有/特效提示
- **按钮风格**：圆角矩形，金色渐变边框，hover时发光效果，重要操作带脉动动画
- **字体**：标题使用 Playfair Display（衬线体，时尚杂志感），正文使用 DM Sans（现代几何体）
- **布局风格**：左侧固定导航栏 + 顶部状态栏 + 主内容区卡片式布局
- **图标风格**：Lucide 图标，线条风格，搭配金色/品红主题色
- **特效**：卡片悬浮微动效，页面切换淡入动画，稀有物品流光粒子效果

### 4.3 响应式设计
- 桌面优先设计，1920x1080为主视口
- 平板端：导航栏折叠为汉堡菜单，卡片网格从4列调整为2列
- 移动端：底部标签导航，单列卡片布局，走秀页面横向滑动

### 4.4 3D场景指引
- 时装秀走秀场景：等距视角T台，观众席背景虚化，聚光灯跟随模特
- 工作室布局：45度等距视角，设施3D模型，可缩放旋转
- 服装预览：360度旋转展示台，流光/星空粒子系统
- 光照：主聚光灯+环境光+RGB边缘光，营造舞台氛围
